## Description

Desafio proposto pela WAProject

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Production
$ docker-compose up -d

# API documentation (Swagger)
$ localhost:3000/api

# development
$ npm run start

# watch mode
$ npm run start:dev

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
